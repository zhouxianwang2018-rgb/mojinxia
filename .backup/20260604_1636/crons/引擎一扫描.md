执行早期突破三阶段扫描，筛选STAGE2/STAGE3候选标的，写入策略文件 candidate_pool.engine1。

## Step 0: 加载策略链

```bash
cat ~/.hermes/cron/state/moni/strategy_index.json
TODAY=$(date +%Y-%m-%d)
STRATEGY_FILE=~/.hermes/cron/state/moni/strategies/$TODAY.json
cat $STRATEGY_FILE
```

1. 确认 trading_day == 今天
2. 读取 risk_state.level：
   - emergency → 跳过扫描，写一句话到策略文件 execution_log，终止
   - cautious/defensive → 继续扫描但标注限制
3. 读取 per_stock：can_buy=false → 从候选剔除

## Step 1: 运行扫描

```bash
python3 ~/.hermes/scripts/early_breakout_scanner.py --watchlist --json 2>/dev/null
```

## Step 2: 写入策略文件

```bash
python3 -c "
import json, subprocess, datetime

TODAY = '$(date +%Y-%m-%d)'
STRATEGY_FILE = f'$HOME/.hermes/cron/state/moni/strategies/{TODAY}.json'

with open(STRATEGY_FILE) as f:
    d = json.load(f)

# 从扫描脚本获取结果
result = subprocess.run(['python3', '$HOME/.hermes/scripts/early_breakout_scanner.py', '--watchlist', '--json'], capture_output=True, text=True)
scan = json.loads(result.stdout) if result.stdout else {'stage3': [], 'stage2': []}

# 过滤 can_buy=false 和止损观察标的
per_stock = d.get('current_strategy', {}).get('per_stock', {})
stage3_filtered = []
stage2_filtered = []
for s in scan.get('stage3', []):
    code = s.get('code', '')
    ps = per_stock.get(code, {})
    if ps.get('can_buy', True) and not ps.get('stop_loss_triggered'):
        stage3_filtered.append(s)
for s in scan.get('stage2', []):
    code = s.get('code', '')
    ps = per_stock.get(code, {})
    if ps.get('can_buy', True) and not ps.get('stop_loss_triggered'):
        stage2_filtered.append(s)

# 写入 candidate_pool.engine1
if 'candidate_pool' not in d['current_strategy']:
    d['current_strategy']['candidate_pool'] = {}
d['current_strategy']['candidate_pool']['engine1'] = {
    'stage3': stage3_filtered,
    'stage2': stage2_filtered,
    'scanned_at': datetime.datetime.now().isoformat(),
    'total_scanned': len(scan.get('stage3',[])) + len(scan.get('stage2',[])),
    'filtered_out': (len(scan.get('stage3',[])) + len(scan.get('stage2',[]))) - (len(stage3_filtered) + len(stage2_filtered))
}

# 记录到 execution_log
d['execution_log'].append({
    'time': datetime.datetime.now().isoformat(),
    'cron': '引擎一扫描',
    'action': f\"STAGE3={len(stage3_filtered)} STAGE2={len(stage2_filtered)} 已写入candidate_pool.engine1\",
    'scenario_matched': 'null',
    'deviation': 'null'
})

with open(STRATEGY_FILE, 'w') as f:
    json.dump(d, f, indent=2, ensure_ascii=False)

print(f\"✅ engine1 written: STAGE3={len(stage3_filtered)} STAGE2={len(stage2_filtered)}\")
"
```
