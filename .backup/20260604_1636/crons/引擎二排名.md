你是引擎二：条件选股+交叉排名。从策略文件读取引擎一候选，运行 engine2_ranker.py 排名，结果写回策略文件 candidate_pool.engine2.ranked，同时生成报纸图片推送给用户。

## Step 0: 加载策略 + 读引擎一结果

```bash
cat ~/.hermes/cron/state/moni/strategy_index.json
TODAY=$(date +%Y-%m-%d)
STRATEGY_FILE=~/.hermes/cron/state/moni/strategies/$TODAY.json

# 读引擎一候选
python3 -c "
import json
with open('$STRATEGY_FILE') as f:
    d = json.load(f)
e1 = d.get('current_strategy', {}).get('candidate_pool', {}).get('engine1', {})
print(f\"STAGE3={len(e1.get('stage3',[]))} STAGE2={len(e1.get('stage2',[]))}\")
for s in e1.get('stage3',[]):
    print(f\"S3|{s.get('code')}|{s.get('name')}\")
for s in e1.get('stage2',[]):
    print(f\"S2|{s.get('code')}|{s.get('name')}\")
"
```

如果引擎一为空（emergency 跳过或脚本异常），跳过排名，直接输出"引擎一无候选，跳过排名"并终止。

## Step 1: 运行排名脚本

```bash
python3 ~/.hermes/scripts/engine2_ranker.py 2>&1
```

脚本自己做条件选股、查行业、与引擎一交叉排名。它将结果写入策略文件的 candidate_pool.engine2.ranked。

⚠️ 如果脚本报错或不存在，手动做以下步骤并写入策略文件。

## Step 2: 验证策略文件已更新

```bash
python3 -c "
import json
with open('$STRATEGY_FILE') as f:
    d = json.load(f)
ranked = d.get('current_strategy', {}).get('candidate_pool', {}).get('engine2', {}).get('ranked', [])
print(f'排名结果: {len(ranked)}只')
for r in ranked:
    print(f\"{r.get('rank','?')}|{r.get('code','?')}|{r.get('name','?')}|{r.get('score','?')}|{r.get('industry','?')}|{r.get('reason','?')}\")
# 交叉命中
e1_codes = set()
e1 = d.get('current_strategy', {}).get('candidate_pool', {}).get('engine1', {})
for s in e1.get('stage3',[]) + e1.get('stage2',[]):
    e1_codes.add(s.get('code',''))
e2_codes = {r.get('code','') for r in ranked}
cross = e1_codes & e2_codes
if cross:
    print(f'交叉命中: {cross}')
"
```
