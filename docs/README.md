# 摸金虾 · 模拟盘交易系统

> v1.0.1 · 2026-06-03

A 股全自动模拟盘交易系统。七道 CRON 接力流水线覆盖 09:00→15:05 全交易日，策略文件驱动状态机，双引擎选股 + 三体联动，五级硬编码风控。
