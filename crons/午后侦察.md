## Step 1: 快速侦察（3个检查，各2分钟）

### 检查A：持仓跳空
用 moni_engine.py 查每只持仓的现价，对比上午收盘价：
- 跳空 >2% → 🚨
- 跌破 stop_loss_price → 🔴 标注 "stop_loss_breached"

### 检查B：板块异动
用 mx_query 查半导体板块涨跌幅：
- 上午 vs 当前 → 变化 >1.5% → 🚨
- 板块暴跌 >3% → 标注 "sector_crash"

### 检查C：午间公告
用 mx_news 搜索 "半导体 公告 午间"，重大利好/利空记录。

## Step 2: 写入结构化市场快照（核心输出）

必须通过 execute_code 写入，格式固定：

```python
from hermes_tools import read_file
import json, os
from datetime import datetime

TODAY = datetime.now().strftime('%Y-%m-%d')
STRATEGY_FILE = os.path.expanduser(f'~/.hermes/cron/state/moni/strategies/{TODAY}.json')

with open(STRATEGY_FILE) as f:
    strategy = json.load(f)

# 用 Step 1 侦察的实际数据填充
strategy['market_snapshot'] = {
    'written_by': '午后侦察',
    'written_at': datetime.now().isoformat(),
    'indices': {
        'sse': {'price': 0000, 'change_pct': 0.00},      # ← 上证指数
        'star50': {'price': 0000, 'change_pct': 0.00},    # ← 科创50
        'csi300': {'price': 0000, 'change_pct': 0.00},    # ← 沪深300
        'gem': {'price': 0000, 'change_pct': 0.00}        # ← 创业板
    },
    'sectors': {},   # ← 如 {"半导体": {"change": "+1.03", "flow": "-48亿", "trend": "V型反弹"}, ...}
    'alerts': [],    # ← 如 [{"level": "🚨", "type": "sector_reversal", "detail": "半导体-1.85%→+1.03%"}]
    'midday_news': [],  # ← 午间重大公告/政策
    'bias': '偏多/中性/偏空',  # ← 综合判断
    'recommendation': ''  # ← 一句话：尾盘重点关注什么
}

with open(STRATEGY_FILE, 'w') as f:
    json.dump(strategy, f, indent=2, ensure_ascii=False)

print(f"✅ market_snapshot 已写入策略文件")
```
